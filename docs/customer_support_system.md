# Customer Support System

## Overview
This document outlines a customer support system that includes both chat functionality and a ticket management system.

## Features
1. **Chat System**
   - Real-time messaging
   - User authentication
   - Chat history

2. **Ticket Management System**
   - Ticket creation and tracking
   - Automated responses
   - Ticket assignment to support agents

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Socket.io (for chat functionality)
- HTML/CSS/JavaScript (for the frontend)

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/xpress09/Jt-express-
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
